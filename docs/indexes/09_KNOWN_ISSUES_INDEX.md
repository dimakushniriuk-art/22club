## Stato sintetico per priorità operativa

| Stato             | Significato operativo                                                             | Azione consigliata                           |
| ----------------- | --------------------------------------------------------------------------------- | -------------------------------------------- |
| **aperto**        | issue reale con impatto alto/critico                                              | trattare come backlog prioritario            |
| **mitigato**      | rischio ridotto ma non chiuso                                                     | evitare regressioni, completare convergenza  |
| **da verificare** | manca evidenza conclusiva runtime                                                 | validare su ambiente reale con matrix mirata |
| **risolto**       | chiuso nelle fonti / nel perimetro dichiarato (es. allineamento overlap UI staff) | —                                            |
